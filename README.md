# Network Architecture HLD - Presentation

A professional HTML/CSS/JS slide deck presentation for Network Architecture High-Level Design.

## Features

- ✅ 10 slides covering complete network architecture
- ✅ Interactive navigation (keyboard arrows, buttons, URL hash routing)
- ✅ SVG network topology diagram
- ✅ Print-friendly (export to PDF via browser print)
- ✅ Responsive design

## Viewing Locally

Simply open `index.html` in any modern web browser.

## Navigation

- **Arrow Keys**: Navigate between slides
- **Home/End**: Jump to first/last slide
- **Spacebar**: Next slide
- **URL Hash**: Direct link to slides (`#slide-1`, `#slide-2`, etc.)

## Files

- `index.html` - Main presentation file
- `styles.css` - Styling and layout
- `deck.js` - Navigation controller
